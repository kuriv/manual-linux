# apt

执行下面的命令，列出所有可更新的软件清单。

```
sudo apt update
```

执行下面的命令，升级所有可更新的软件包。

```
sudo apt upgrade
```

执行下面的命令，升级指定的软件包。

```
sudo apt upgrade vim
```

执行下面的命令，安装指定的软件包。

```
sudo apt install vim
```

执行下面的命令，删除指定的软件包。

```
sudo apt remove vim
```

执行下面的命令，显示指定的软件包信息。

```
sudo apt show vim
```

执行下面的命令，清理不再使用的依赖和库文件。

```
sudo apt autoremove
```

执行下面的命令，列出所有已安装的软件包。

```
sudo apt list --installed
```
